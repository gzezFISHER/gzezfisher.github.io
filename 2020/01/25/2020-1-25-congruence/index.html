同余方程

扩展欧几里德算法

问题

求解形如 $a\cdot x + b\cdot y = c$ ( $a, b, c$ 为参数)的方程。

前置知识

裴蜀定理

对于正整数 $a, b$, $a\cdot x + b\cdot y$ 是 $d = (a, b)$ 的倍数。
下证:
令 $d\cdot k_1 = a, d\cdot k_2 = b$
则 $a\cdot x + b\cdot y = (k_1 + k_2)\cdot d$

有了裴蜀定理，就可以知道方程 $a\cdot x + b\cdot y = c$ 是否有解了。

解法

  1. $d\nmid c$
    由裴蜀定理，该方程无整数解。
  2. $d\mid c$ 且 $d = c$
    考虑对欧几里德算法(辗转相除法)进行拓展。
    即我们已经知道 $b\cdot x + (a \mod b)\cdot y = d$ 的解 $x_0, y_0$，要通过 $x_0, y_0$ 算出 $a\cdot x + b\cdot y = c$ 的解 $x, y$。
    由 $b\cdot x_0 + (a \mod b)\cdot y_0 = d$
    得 $b\cdot x_0 + a\cdot y_0 - b\cdot \lfloor \frac{a}{b} \rfloor \cdot y_0 = d$
    即 $a\cdot y_0 + b\cdot (x_0 - \lfloor \frac{a}{b} \rfloor \cdot y_0) = d$
    则有
    $$ \left\{ \begin{align*} x &= y_0\\ y &= x_0 - \lfloor \frac{a}{b} \rfloor \cdot y_0 \end{align*} \right. $$
    该方程的解并不只有一组。 假设现在以求得一组解 $x_0, y_0$, 存在另外的解 $x, y$，有 $a\cdot (x_0 - x) = b\cdot (y - y_0)$ 则 $\frac{b}{d} \mid x_0 - x$ 不妨设 $x = x_0 + k\cdot \frac{b}{d}$ ($k$ 为整数) 则 $x_0\cdot a + k\cdot \frac{b}{d}\cdot a + y\cdot b = d$ 又 $x_0\cdot a + y_0\cdot b = d$ 所以 $y = y_0 - k\cdot \frac{a}{d}$ 即对于所有整数 $k$，$x = x_0 + k\cdot \frac{b}{d}, y = y_0 - k\cdot \frac{a}{d}$ 为原方程的一组解。
  3. $d\mid c$ 且 $d \not = c$
    令 $d\cdot k = c$
    先解出 $a\cdot x + b\cdot y = d$ 的解 $x_0, y_0$
    显然原方程的解即为 $x = k\cdot x_0, y = k\cdot y_0$

代码

乘法逆元

问题

若 $a\cdot x \equiv 1 (\mod p)$，称 $x$ 为 $a$ 在模 $p$ 意义下的逆元，记作 $a^{-1}$。

应用

取模意义下的除法 $\frac{a}{b} = a\cdot b^{-1}$

解法

扩展欧几里德

令 $-k\cdot p + 1 = a\cdot x$
则有 $a\cdot x + k\cdot p = 1$
即可用扩展欧几里德求解。

注意

当且仅当 $(a, k) = 1$ 时可解。

费马小定理

对于质数 $p$ 和整数 $a$ ($p \nmid a$)，有 $a^{p-1}\equiv 1(\mod p)$。
下证:
集合 $S = {a\mod p, 2\cdot a\mod p, \cdots, (p - 1)\cdot a\mod p}$
假设集合中有两个元素相同，即存在 $i, j$ ($i \not = j$ 且 $1\le i, j\le p - 1$) 使得 $i\cdot a\equiv j\cdot a(\mod p)$
则 $p \mid (i - j)\cdot a$
又 $p$ 是质数且 $p \mid a$
所以 $p \mid (i - j)$，矛盾
所以 $S$ 的元素两两不同
即 $S = {1, 2, \cdots, p - 1}$
$\prod_{i = 1}^{p - 1} S_i\equiv \prod_{i = 1}^{p - 1} i\cdot a(\mod p)$
即 $(p - 1)!\equiv (p - 1)!\cdot a^{p - 1}(\mod p)$
所以 $a^{p - 1}\equiv 1(\mod p)$

由费马小定理，$a^{p - 1}\equiv 1(\mod p)$
则 $a\cdot a^{p - 2}\equiv 1(\mod p)$
即 $a^{-1} = a^{p - 2}$

注意

需满足 $p$ 是质数且 $p \nmid a$。
